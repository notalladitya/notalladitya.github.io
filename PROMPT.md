## Technical Stack
- **Package Manager** pnpm, pnpx for high performance
- **Next.js** (App Router) for static site generation
- **Framer Motion** (or Motion) for smooth, spring-physics animations
- **Tailwind CSS** for styling with custom design tokens
- Inside /portfolio 
- Static export enabled for easy deployment to GitHub Pages
- TypeScript for type safety
- Lucide React for icons
